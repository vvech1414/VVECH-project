## 1. Что делает фича

- **Шаг:** 5
- **Экран:** Drill-down в саммари
- **Действия пользователя:** Кликает на строку «Готов»
- **Данные системы:** selectedPatientId
- **Happy path:** Переходит на D02 c анимацией 200ms

## 2. Контекст эпика

- **Initiative flow:** Врач открывает кокпит → видит сегодняшнюю очередь приёмов → видит готовность каждого пациента до открытия карточки → кликает на готового пациента
- **JTBD functional:** За 10 секунд понять, кто из пациентов готов к приёму
- **JTBD emotional:** Уверенность в начале рабочего дня
- **Состояние эпика в прототипе:** Готово
- **Комментарий из реестра:** PatientList (D01) — 6 строк mock-данных

## 3. Применимые гардрейлы

> Поверхность DOC — out of scope for this prototype repo per prototype CLAUDE.md, спек сгенерирован как референс.

- **Audit-логирование (N5)** — открытие профиля и документа, запросы, редактирование заметок логируются.

- **AI consultative only** — никаких diagnosis/treatment/prescription формулировок.
- **Copy на русском** — спокойный, практичный тон.

**Тон:** спокойный, практичный, не алармистский. Короткие лейблы, поддерживающий helper-текст.

**Хорошие примеры:**

**Избегать:**
