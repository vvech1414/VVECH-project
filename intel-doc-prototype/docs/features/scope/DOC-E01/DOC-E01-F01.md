## 3. Применимые гардрейлы

> Поверхность DOC — out of scope for this prototype repo per prototype CLAUDE.md, спек сгенерирован как референс.

- **Doctor cockpit реализуется не в этом репо** — допускается только как симулированное входящее состояние (запрос, план, уведомление).
- **Role-based access (N6)** — роль врача и скоуп отделения проверяются на каждом чтении.
- **Audit-логирование (N5)** — открытие профиля и документа, запросы, редактирование заметок логируются.
- **Прототип, не продакшн** — mock-state, без реальных API/EHR/MIS интеграций.
- **AI consultative only** — никаких diagnosis/treatment/prescription формулировок.
- **Copy на русском** — спокойный, практичный тон.
