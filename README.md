# IntelDoc Prototype

Учебный выпускной проект. Прототип продукта **IntelDoc** для пилотного запуска в ЭНЦ — три ролевых интерфейса в одном React-приложении:

1. **Пациент (mobile)** — подготовка к приёму, загрузка анализов, OCR-проверка, план обследования.
2. **Врач (web)** — очередь приёма и структурированная сводка по пациенту.
3. **Администратор клиники (web)** — KPI и аудит доступа.

## Статус

Документация и бриф готовятся. Прототип будет жить в `react-prototype/inteldoc_patient_prototype/`.
